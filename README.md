# **ETABS, SAP2000, SAFE & CSiBridge OAPI Examples CSiAmerica Software API Examples**
### CSI America API Examples in C#, VB.NET and Python for Structural Engineering Automation

Welcome to **CSiAPIExamples** — a beginner-friendly, practical collection of examples for using the **CSI America OAPI (Open Application Programming Interface)**.

This repository provides working code examples for automating:

- ETABS API
- SAP2000 API
- SAFE API
- CSiBridge API
- Perform3D API

If you are a **structural engineer learning programming**, this repository is designed for you.

# 🚀 What Is CSI OAPI?

The **CSI OAPI (Open API)** allows you to control CSI software programmatically using:

- C#
- VB
- Excel Visual basic advance
- Python
- C++
- MATLAB

With OAPI, you can:

- Automatically create structural models  
- Assign materials and sections  
- Define loads and combinations  
- Run analysis  
- Extract results  
- Read and write database tables  
- Automate repetitive modeling tasks  

This repository contains simple, clean, beginner-focused examples to help you get started.

# 🎯 What You Will Learn

Using these examples, you will learn how to:

- Connect to ETABS via C#
- Attach to a running SAP2000 instance
- Start SAFE programmatically
- Create nodes and frame elements
- Assign properties and materials
- Run structural analysis from code
- Extract analysis results
- Access CSI database tables
- Automate engineering workflows

These examples are intentionally simple so you can clearly understand the CSI API workflow.

# **Getting Started**

## **Prerequisites**

- **CSiAmerica Software**: Ensure you have a valid installation of supported CSiAmerica software (e.g., SAP2000, ETABS, SAFE, CSiBridge, Perform3D).
- **Programming Language**: Knowledge of the programming language supported by the API (Python, Visual Basic, C#, C++, MATLAB).
- **Development Environment**:
  - We recommend using **Visual Studio** or **Visual Studio Code** for running and debugging the provided examples.


## **Installing Visual Studio or Visual Studio Code**

### **Visual Studio Code**
1. Download **Visual Studio Code (VS Code)** from the official site: [https://code.visualstudio.com/](https://code.visualstudio.com/)

### **Installing Visual Studio (For C#, Visual Basic, and C++)**
1. Download **Visual Studio** from [https://visualstudio.microsoft.com/](https://visualstudio.microsoft.com/).
2. During the installation process, you’ll need to select the following workloads:
   - **.NET desktop development**: For coding in **C#** and **Visual Basic**.
   - **Desktop development with C++**: For coding in **C++**.
3. After installation, Visual Studio will have the necessary compilers and libraries for **C#**, **Visual Basic**, and **C++** development.
